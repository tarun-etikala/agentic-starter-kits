# AutoGen Agents

Agent templates built with [AutoGen](https://microsoft.github.io/autogen/) (Microsoft).

## Available Agents

| Agent | Description |
|-------|-------------|
| [mcp_agent](mcp_agent/) | AssistantAgent with MCP tools over SSE (e.g. churn deployment, `add`/`sub`), FastAPI `/chat/completions` |
